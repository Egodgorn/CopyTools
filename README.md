<br># CopyTools</br>
<br>Это мой тулз, который нужен был мне для работы</br>
Позволяет создавать большое количество одинаковых строк,с разницей в конечной цифре
Пример:
  <br>  TextBox1
  <br>  TextBox2
  <br>  TextBox3
  <br>  TextBox4
  <br>  TextBox5
 <br>Так же можно создавать копии с цифро внутри слова</br>
 Пример:
  tr("TextBox1")</br>
  tr("TextBox2")</br>
  tr("TextBox3")</br>
  tr("TextBox4")</br>
  tr("TextBox5")</br>
