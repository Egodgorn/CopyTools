# CopyTools
Это мой тулз, который нужен был мне для работы
Позволяет создавать большое количество одинаковых строк,с разницей в конечной цифре
Пример:
  TextBox1
  TextBox2
  TextBox3
  TextBox4
  TextBox5
 Так же можно создавать копии с цифро внутри слова
 Пример:
  tr("TextBox1")
  tr("TextBox2")
  tr("TextBox3")
  tr("TextBox4")
  tr("TextBox5")
